Функция \(\displaystyle f\left( x \right):X\to \mathbb{R}\) возрастает (убывает) в точке \(\displaystyle x={{x}_{0}}\in X\), если существует некоторая окрестность этой точки, в которой


\(\displaystyle \left\{ \begin{matrix} f\left( x \right)\,>\,f\left( {{x}_{0}} \right)\text{ }\text{ }x\,>\,{{x}_{0}}; \\ f\left( x \right)\,<\,f\left( {{x}_{0}} \right)\text{ }\text{ }x\,<\,{{x}_{0}}, \\ \end{matrix} \right.\) \(\displaystyle \left( \left\{ \begin{matrix} f\left( x \right)\,<\,f\left( {{x}_{0}} \right)\text{ }\text{ }x\,>\,{{x}_{0}}; \\ f\left( x \right)\,>\,f\left( {{x}_{0}} \right)\text{ }\text{ }x\,<\,{{x}_{0}}. \\ \end{matrix} \right. \right)\)


Если обозначить \(\displaystyle \Delta x=x-{{x}_{0}}\), \(\displaystyle \Delta f=f\left( x \right)-f\left( {{x}_{0}} \right)\), то определение можно переформулировать следующим образом: функция \(\displaystyle f\left( x \right)\) возрастает (убывает) в точке \(\displaystyle x\,=\,{{x}_{0}}\), если при \(\displaystyle \Delta x\,\ne \,0\) \(\displaystyle \frac{\Delta f}{\Delta x}\,>\,0\) (\(\displaystyle \frac{\Delta f}{\Delta x}\,<\,0\)). Точка, в которой функция возрастает (убывает), называется точкой возрастания (убывания) функции \(\displaystyle f\left( x \right)\).


Функция \(\displaystyle f\left( x \right):X\to \mathbb{R}\) называется:

неубывающей на множестве \(\displaystyle A\), если

\(\displaystyle \forall {{x}_{1}},{{x}_{2}}\in A\bigcap X\ \ {{x}_{1}}\ne {{x}_{2}}\ \Rightarrow \ \frac{f\left( {{x}_{1}} \right)-f\left( {{x}_{2}} \right)}{{{x}_{1}}-{{x}_{2}}}\ge 0\);

– невозрастающей на множестве \(\displaystyle A\), если

\(\displaystyle \forall {{x}_{1}},{{x}_{2}}\in A\bigcap X\ \ {{x}_{1}}\ne {{x}_{2}}\ \Rightarrow \ \frac{f\left( {{x}_{1}} \right)-f\left( {{x}_{2}} \right)}{{{x}_{1}}-{{x}_{2}}}\le 0\);

– возрастающей на множестве \(\displaystyle A\), если

\(\displaystyle \forall {{x}_{1}},{{x}_{2}}\in A\bigcap X\ \ {{x}_{1}}\ne {{x}_{2}}\ \Rightarrow \ \frac{f\left( {{x}_{1}} \right)-f\left( {{x}_{2}} \right)}{{{x}_{1}}-{{x}_{2}}}>0\);

– убывающей на множестве \(\displaystyle A\), если

\(\displaystyle \forall {{x}_{1}},{{x}_{2}}\in A\bigcap X\ \ {{x}_{1}}\ne {{x}_{2}}\ \Rightarrow \ \frac{f\left( {{x}_{1}} \right)-f\left( {{x}_{2}} \right)}{{{x}_{1}}-{{x}_{2}}}<0\).

Неубывающие, невозрастающие, убывающие и возрастающие функции объединяются общим наименованием монотонные функции; убывающие и возрастающие функции называются также строго монотонными.

Замечание. В определении предела \(\displaystyle \underset{x\to {{x}_{0}}}{\mathop{\lim }}\,f\left( x \right)\) нет никаких условий на значение \(\displaystyle f\left( {{x}_{0}} \right)\), функция может быть даже не определена в точке \(\displaystyle {{x}_{0}}\). Поэтому если две функции \(\displaystyle f\left( x \right)\) и \(\displaystyle g\left( x \right)\)совпадают в некоторой проколотой окрестности точки \(\displaystyle {{x}_{0}}\), то либо обе функции не имееют предела при \(\displaystyle x\to {{x}_{0}}\), либо \(\displaystyle \underset{x\to {{x}_{0}}}{\mathop{\lim }}\,f\left( x \right)=\underset{x\to {{x}_{0}}}{\mathop{\lim }}\,g\left( x \right)\). Аналогичное утверждение справедливо и для пределов \(\displaystyle \underset{x\to {{x}_{0}}+}{\mathop{\lim }}\,f\left( x \right)\), \(\displaystyle \underset{x\to {{x}_{0}}-}{\mathop{\lim }}\,f\left( x \right)\) на значение предела никак не влияет.


Пример 3.1. Докажем по определению, что \(\displaystyle \underset{x\to 3}{\mathop{\lim }}\,\left( 5x+1 \right)=16\).


\(\displaystyle \forall \varepsilon >0\,\,\exists \delta =\frac{\varepsilon }{5}\) \(\displaystyle \forall x\in X\,\,\,0<\left| x-3 \right|<\delta \,\,\Rightarrow \,\,\left| \left( 5x+1 \right)-16 \right|=5\left| x-3 \right|<5\delta =5\frac{\varepsilon }{5}=\varepsilon \)


Пример 3.2. Пусть \(\displaystyle f\left( x \right)=\left\{ \begin{matrix} 1,x\ne 0, \\ 0,x=0. \\ \end{matrix} \right.\) Докажем по определению, что \(\displaystyle \underset{x\to 0}{\mathop{\lim }}\,f\left( x \right)=1\).


В любой проколотой окрестности 0 \(\displaystyle f\left( x \right)=1\), следовательно, \(\displaystyle \forall \varepsilon >0\,\,\exists \delta =\varepsilon \) \(\displaystyle \forall x\in X\,\,\,0<\left| x \right|<\delta \,\,\Rightarrow \,\,\left| f\left( x \right)-1 \right|=0<\varepsilon \)
