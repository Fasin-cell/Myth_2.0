  1. Равномерная сходимость несобственных интегралов

Несобственный интеграл называется равномерно сходящимся по параметру \(\displaystyle \gamma \) на \(\displaystyle \Gamma \), если он сходится на \(\displaystyle \Gamma \) и выполняется одно из следующих эквивалентных условий:

1) x \(\displaystyle \forall \varepsilon >0\) x \(\displaystyle \exists {{A}_{\varepsilon }}\ge a\) x \(\displaystyle \forall A\in \left( {{A}_{\varepsilon }},\mathbf{B} \right)\)

x \(\displaystyle \forall \gamma \in \Gamma \) x \(\displaystyle \left| \int\limits_{A}^{\mathbf{B}}{f(x,\gamma )dx} \right|<\text{ }\!\!\varepsilon\!\!\text{ }\) или x \(\displaystyle \underset{\gamma \in \Gamma }{\mathop{\sup }}\,\) x \(\displaystyle \left| \int\limits_{A}^{\mathbf{B}}{f(x,\gamma )dx} \right|<\text{ }\!\!\varepsilon\!\!\text{ }\),

1) \(\displaystyle \forall \varepsilon >0\) \(\displaystyle \exists {{\eta }_{\varepsilon }}\ge a\) \(\displaystyle \forall \eta \in \left( {{\eta }_{\varepsilon }},\mathbf{B} \right)\) \(\displaystyle \forall \gamma \in \Gamma \) \(\displaystyle \left| \int\limits_{\eta }^{\mathbf{B}}{f(x,\gamma )dx} \right|<\text{ }\!\!\varepsilon\!\!\text{ }\),

2) \(\displaystyle \forall \varepsilon >0\) \(\displaystyle \exists {{\eta }_{\varepsilon }}\ge a\) \(\displaystyle \forall \eta \in \left( {{\eta }_{\varepsilon }},\mathbf{B} \right)\) \(\displaystyle \underset{\gamma \in \Gamma }{\mathop{\sup }}\,\) \(\displaystyle \left| \int\limits_{\eta }^{\mathbf{B}}{f(x,\gamma )dx} \right|<\text{ }\!\!\varepsilon\!\!\text{ }\),

3) \(\displaystyle \underset{\eta \to \mathbf{B}}{\mathop{\lim }}\,\underset{\gamma \in \Gamma }{\mathop{\sup }}\,\left| \int\limits_{\eta }^{\mathbf{B}}{f\left( x,\gamma \right)dx} \right|=0.\)

Замечание 1. Равномерная сходимость интеграла x \(\displaystyle \int\limits_{a}^{\infty }{f(x,\gamma )dx}\), x \(\displaystyle \gamma \in \Gamma \) означает, что наряду с существованием конечных пределов x \(\displaystyle \underset{A\to \infty }{\mathop{\lim }}\,\int\limits_{a}^{A}{f(x,\gamma )dx}\) для каждого фиксированного x \(\displaystyle \gamma \in \Gamma \), существует и конечен предел x \(\displaystyle \underset{A\to \infty }{\mathop{\lim }}\,\underset{\gamma \in \Gamma }{\mathop{\sup }}\,\int\limits_{a}^{A}{f(x,\gamma )dx}\).

Замечание 2. Для несобственных интегралов второго рода аналогично формулируются теоремы о непрерывности, интегрируемости и дифференцируемости по параметру (так как несобственные интегралы второго рода с помощью преобразования переменной x \(\displaystyle x\) сводятся к зависящим от параметра несобственным интегралам первого рода).

Несобственный интеграл x \(\displaystyle \int\limits_{a}^{b}{f\left( x,\gamma \right)dx}\) II-го рода заменой x \(\displaystyle x=\frac{bt+a}{t+1}\) приводится к несобственному интегралу I рода x \(\displaystyle \int\limits_{0}^{+\infty }{f\left( \frac{bt+a}{t+1} \right)\frac{b-a}{{{\left( t+1 \right)}^{2}}}dt}\).

Теорема 14 (критерий равномерной сходимости Коши). Для равномерной по параметру \(\displaystyle \gamma \) на \(\displaystyle \Gamma \) сходимости несобственного интеграла

\(\displaystyle I(\gamma )=\int\limits_{a}^{\mathbf{B}}{f(x,\gamma )dx}\), \(\displaystyle \gamma \in \Gamma \)

необходимо и достаточно, чтобы

\(\displaystyle \forall \varepsilon >0\) \(\displaystyle \exists {{\eta }_{\varepsilon }}\ge a\) \(\displaystyle \forall {\eta }'\in \left( {{\eta }_{\varepsilon }},\mathbf{B} \right)\,\,\,\forall {\eta }''\in \left( {{\eta }_{\varepsilon }},\mathbf{B} \right)\) \(\displaystyle \underset{\gamma \in \Gamma }{\mathop{\sup }}\,\)\(\displaystyle \left| \int\limits_{{{\eta }'}}^{{{\eta }''}}{f(x,\gamma )\,dx} \right|<\text{ }\!\!\varepsilon\!\!\text{ }\).

Следствие (связь с функциональными рядами). Равномерная по \(\displaystyle \gamma \) сходимость интеграла \(\displaystyle I(\gamma )\), \(\displaystyle \gamma \in \Gamma \) эквивалентна равномерной сходимости всех рядов вида \(\displaystyle \sum\limits_{n=0}^{+\infty }{\int\limits_{{{c}_{n}}}^{{{c}_{n+1}}}{f(x,\gamma )dx}}\), где \(\displaystyle \left\{ {{c}_{n}} \right\}\) – произвольная, сходящаяся к \(\displaystyle \mathbf{B}\) последовательность.
